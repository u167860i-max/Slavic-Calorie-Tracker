# Веданіе о Питаниѣ

Приложение для отслеживания калорий и питания с древнерусским интерфейсом.

## Описание

Приложение позволяет вести учет потребляемых калорий, белков, жиров и углеводов. Интерфейс выполнен в стиле древнерусского языка, что придает приложению уникальный характер.

## Возможности

-  **Библиотека продуктов** ("Книга Яствъ") - добавление и редактирование продуктов с информацией о калориях и БЖУ на 100 грамм
-  **Отслеживание приемов пищи** ("Мой Пиръ") - запись продуктов в рацион с указанием количества, типа приема пищи и даты
-  **Статистика по дням** - подсчет общей суммы калорий и БЖУ за выбранный день
-  **Визуализация** - отображение образа в зависимости от количества приемов пищи за день
-  **Выбор даты** - возможность просматривать и редактировать записи за любой день

## Требования

- Python 3.x
- tkinter 
- tkcalendar
- PIL (Pillow)

## Использование

1. **Добавление продукта:**
   - Перейдите на вкладку "Книга Яствъ"
   - Введите название продукта, калории и БЖУ на 100 грамм
   - Нажмите "Внести в Книгу яствъ"

2. **Добавление приема пищи:**
   - Перейдите на вкладку "Мой Пиръ"
   - Выберите продукт из списка
   - Укажите количество в граммах
   - Выберите тип приема пищи (Заутрок, Обед, Вечеря)
   - Выберите дату
   - Нажмите "Вписати яство в пиръ"

3. **Просмотр статистики:**
   - Статистика за выбранный день отображается автоматически в разделе "Итоги за денъ"

4. **Просмотр образа:**
   - Нажмите кнопку "Ведати Образъ" для просмотра образа в зависимости от количества приемов пищи

## Особенности

- При первом запуске автоматически создается база данных с библиотекой из 33 продуктов
- Все данные хранятся локально в файле `calories.db`
- Интерфейс полностью на "древнерусском" языке
- Поддержка редактирования и удаления записей
